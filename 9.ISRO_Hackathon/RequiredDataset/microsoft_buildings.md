Dataset: Microsoft Global ML Building Footprints

Official source:

- GitHub repository (global footprints): https://github.com/microsoft/GlobalMLBuildingFootprints

Notes / access:

- Microsoft provides building footprint vector data (country-level). Download from GitHub or official download links provided in the repository.
- For Earth Engine ingestion, convert to GeoJSON/Shapefile and upload as an asset or process locally and store under `data/raw/osm_buildings/`.
